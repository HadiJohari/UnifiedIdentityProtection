# Unified Identity Protection (UIP)

Official research and platform overview for the Unified Identity Protection (UIP) framework, developed by Hadi Johari.

## Live Website
You can access the live webpage at:
[https://hadijohari.github.io/UnifiedIdentityProtection/](https://hadijohari.github.io/UnifiedIdentityProtection/)

## Overview
UIP is a revolutionary approach to identity security, acting as an Identity-Based Firewall to enforce real-time Zero Trust. It focuses on:
- Stopping lateral movement
- Preventing credential-based attacks (Pass-the-Hash, Session Hijacking, etc.)
- Protecting high-privilege accounts (Admins & NHIs)
- Automated Incident Response
